## Overview

Rows are the actual data records you create within templates. Each row represents a single item or entry in your collection - a specific car, a particular apartment, one job application, etc. Rows store their data as JSON, conforming to the field structure defined by their parent template.

<Info>
If templates are like database tables and fields are like columns, then rows are the actual records - the data itself.
</Info>

## Data Structure

### Database Schema

```sql
CREATE TABLE object_row (
  id            BINARY(16) NOT NULL,
  owner_id      VARCHAR(64) NOT NULL,
  template_id   BINARY(16) NOT NULL,
  display_name  VARCHAR(160) NOT NULL,
  values_json   JSON NOT NULL,
  created_at    TIMESTAMP(3) NOT NULL DEFAULT CURRENT_TIMESTAMP(3),
  updated_at    TIMESTAMP(3) NOT NULL DEFAULT CURRENT_TIMESTAMP(3) ON UPDATE CURRENT_TIMESTAMP(3),
  PRIMARY KEY (id),
  CONSTRAINT fk_row_template
    FOREIGN KEY (template_id) REFERENCES template(id)
      ON DELETE CASCADE,
  KEY ix_row_owner_template_created (owner_id, template_id, created_at)
);
```

<Note>
The `values_json` column stores all field data as a JSON object, enabling dynamic schemas without migrations.
</Note>

### Validation Rules

When creating a row, the service validates:

1. **Template ownership** - Template must exist and belong to the user
2. **Field existence** - Template must have at least one field
3. **Valid keys** - All keys in `values` must match existing field keys
4. **Required fields** - All required fields must be present
5. **Type validation** - Each value must match its field's data type

### Creation Example

```java
public RowResponse create(UUID templateId, CreateRowRequest req) {
    // Verify template exists and belongs to user
    templateRepository.findByIdAndOwnerId(templateId, ownerId())
            .orElseThrow(() -> new ConflictException(
                "Template no encontrado o no pertenece al usuario"
            ));

    // Load field definitions
    List<TemplateFieldEntity> fields = fieldRepository
            .findByTemplateIdOrderByOrderIndexAsc(templateId);
    if (fields.isEmpty()) {
        throw new ConflictException(
            "No puedes crear filas: el template no tiene columnas definidas"
        );
    }

    // Build field lookup map
    Map<String, TemplateFieldEntity> fieldMap = new HashMap<>();
    for (TemplateFieldEntity f : fields) 
        fieldMap.put(f.getFieldKey(), f);

    Map<String, Object> values = req.getValues();

    // Validate no unknown keys
    for (String key : values.keySet()) {
        if (!fieldMap.containsKey(key)) {
            throw new ConflictException(
                "Campo no permitido en values: " + key
            );
        }
    }

    // Validate all required fields are present
    for (TemplateFieldEntity f : fields) {
        if (f.isRequired() && !values.containsKey(f.getFieldKey())) {
            throw new ConflictException(
                "Falta campo obligatorio: " + f.getFieldKey()
            );
        }
    }

    // Validate types
    for (Map.Entry<String, Object> entry : values.entrySet()) {
        String key = entry.getKey();
        Object v = entry.getValue();
        TemplateFieldEntity f = fieldMap.get(key);
        validateType(f, v);
    }

    // Serialize to JSON and save
    String valuesJson = objectMapper.writeValueAsString(values);

    ObjectRowEntity row = new ObjectRowEntity();
    row.setOwnerId(ownerId());
    row.setTemplateId(templateId);
    row.setDisplayName(req.getDisplayName().trim());
    row.setValuesJson(valuesJson);

    ObjectRowEntity saved = rowRepository.save(row);

    // Return response with deserialized values
    Map<String, Object> map = objectMapper.readValue(
        saved.getValuesJson(), 
        Map.class
    );
    return new RowResponse(
        saved.getId(), 
        saved.getDisplayName(), 
        map, 
        saved.getCreatedAt(), 
        saved.getUpdatedAt()
    );
}
```

## Updating Rows (PATCH)

Rows support partial updates using PATCH semantics - you only need to send the fields you want to change:

```java
public RowResponse patch(UUID templateId, UUID rowId, PatchRowRequest req) {
    // Verify template and row exist
    templateRepository.findByIdAndOwnerId(templateId, ownerId())
            .orElseThrow(() -> new ConflictException(
                "Template no encontrado o no pertenece al usuario"
            ));

    ObjectRowEntity row = rowRepository
            .findByIdAndOwnerIdAndTemplateId(rowId, ownerId(), templateId)
            .orElseThrow(() -> new ConflictException("Row no encontrada"));

    // Load current values
    Map<String, Object> current = objectMapper.readValue(
        row.getValuesJson(), 
        Map.class
    );

    // Apply partial update
    Map<String, Object> patchValues = req.getValues();
    if (patchValues != null) {
        // Validate unknown keys
        for (String key : patchValues.keySet()) {
            if (!fieldMap.containsKey(key)) {
                throw new ConflictException(
                    "Campo no permitido en values: " + key
                );
            }
        }
        
        // Validate types for patched fields
        for (Map.Entry<String, Object> entry : patchValues.entrySet()) {
            TemplateFieldEntity f = fieldMap.get(entry.getKey());
            validateType(f, entry.getValue());
        }

        // Merge patch into current values
        current.putAll(patchValues);
    }

    // Validate all required fields after merge
    for (TemplateFieldEntity f : fields) {
        if (f.isRequired()) {
            if (!current.containsKey(f.getFieldKey()) || 
                current.get(f.getFieldKey()) == null) {
                throw new ConflictException(
                    "Falta campo obligatorio: " + f.getFieldKey()
                );
            }
        }
    }

    // Update display name if provided
    if (req.getDisplayName() != null) {
        String dn = req.getDisplayName().trim();
        if (dn.isEmpty()) 
            throw new ConflictException(
                "displayName no puede quedar vacío"
            );
        row.setDisplayName(dn);
    }

    // Save updated values
    row.setValuesJson(objectMapper.writeValueAsString(current));
    ObjectRowEntity saved = rowRepository.save(row);

    Map<String, Object> out = objectMapper.readValue(
        saved.getValuesJson(), 
        Map.class
    );
    return new RowResponse(
        saved.getId(), 
        saved.getDisplayName(), 
        out, 
        saved.getCreatedAt(), 
        saved.getUpdatedAt()
    );
}
```

## Listing Rows

Rows are retrieved in reverse chronological order (newest first):

```java
public List<RowResponse> list(UUID templateId) {
    templateRepository.findByIdAndOwnerId(templateId, ownerId())
            .orElseThrow(() -> new ConflictException(
                "Template no encontrado o no pertenece al usuario"
            ));

    return rowRepository
            .findByOwnerIdAndTemplateIdOrderByCreatedAtDesc(ownerId(), templateId)
            .stream()
            .map(r -> {
                Map<String, Object> map;
                try { 
                    map = objectMapper.readValue(r.getValuesJson(), Map.class); 
                } catch (Exception e) { 
                    map = new HashMap<>(); 
                }
                return new RowResponse(
                    r.getId(), 
                    r.getDisplayName(), 
                    map, 
                    r.getCreatedAt(), 
                    r.getUpdatedAt()
                );
            })
            .toList();
}
```

## Deleting Rows

Rows can be deleted with cascading effects on their associated logs:

```java
public void delete(UUID templateId, UUID rowId) {
    templateRepository.findByIdAndOwnerId(templateId, ownerId())
            .orElseThrow(() -> new ConflictException(
                "Template no encontrado o no pertenece al usuario"
            ));

    ObjectRowEntity row = rowRepository
            .findByIdAndOwnerIdAndTemplateId(rowId, ownerId(), templateId)
            .orElseThrow(() -> new ConflictException("Row no encontrada"));

    rowRepository.delete(row);
}
```

<Warning>
Deleting a row also deletes all associated log entries due to the `ON DELETE CASCADE` constraint.
</Warning>

## Display Names

The `display_name` field provides a human-readable identifier for each row:

- Maximum length: 160 characters
- Required for all rows
- Can be updated via PATCH
- Commonly used for quick identification in lists (e.g., "Toyota Corolla 2020", "123 Main St Apartment")

## What is Bitácora Universal?

Bitácora Universal is a personal notebook application where you can:

- **Create Templates**: Define categories for anything you want to track (cars, cities, books, workouts)
- **Define Custom Fields**: Add columns with different data types (text, numbers, dates, booleans, selects)
- **Add Records**: Create unlimited rows of data within each template
- **Track with Logs**: Add historical entries with scores (0-10), comments, and dates

## Example Use Cases

<AccordionGroup>
  <Accordion title="Car Collection">
    Create a "Cars" template with fields like Brand, Model, Year, Fuel Type, and Color. Add each car as a row, then log maintenance records with scores for condition, mileage updates, and service notes.
  </Accordion>
  
  <Accordion title="Restaurant Reviews">
    Build a "Restaurants" template with Name, Cuisine Type, Location, and Price Range fields. Add restaurants as rows and log each visit with a score (1-10), what you ordered, and your experience notes.
  </Accordion>
  
  <Accordion title="Travel Destinations">
    Design a "Cities" template with fields for Country, Population, Best Season, and Must-Visit Places. Track your visits with logs containing ratings, travel dates, and memorable experiences.
  </Accordion>
  
  <Accordion title="Workout Tracker">
    Create an "Exercises" template with Exercise Name, Muscle Group, Equipment, and Difficulty Level. Log each workout session with performance scores, weight/reps, and progress notes.
  </Accordion>
</AccordionGroup>

## Technology Stack

Bitácora Universal is built with modern technologies:

**Backend**
- Java 21
- Spring Boot 4
- Spring Security + JWT authentication
- Spring Data JPA
- MySQL 8
- Flyway migrations

**Frontend**
- React 19
- TypeScript
- Vite
- Tailwind CSS

**Infrastructure**
- Docker + Docker Compose
- RESTful API design
